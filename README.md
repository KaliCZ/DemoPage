# DemoPage

A fun project for my personal website hosted at [kalandra.tech](https://kalandra.tech). Serves as a showcase of my abilities as well as a playground for me.

## What's here

- [**Home**](https://kalandra.tech) — intro and navigation
- [**About Me**](https://kalandra.tech/about) — career timeline, manifesto, links
- [**Project**](https://kalandra.tech/project) — live roadmap tracking the build progress of this site

## Tech stack (v1)

- Cloudflare Pages (hosting)
- GitHub Actions (CI/CD)
